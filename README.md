Questo progetto è una web app per la gestione di un portafoglio digitale, pensata per offrire agli utenti un modo semplice e sicuro per registrarsi, accedere e visualizzare il proprio wallet. L’applicazione è divisa in due parti: un backend sviluppato con Spring Boot e un frontend realizzato in React. I due sistemi comunicano tramite API REST e utilizzano un sistema di autenticazione basato su JWT, che permette di proteggere le operazioni sensibili e distinguere tra utenti normali e amministratori.
Gli utenti possono creare un account, effettuare il login e accedere alla propria dashboard personale, dove trovano il saldo del wallet e le informazioni principali. Gli amministratori, invece, vengono reindirizzati automaticamente a un’area dedicata alla gestione degli utenti. Il token JWT viene salvato nel browser e viene inviato automaticamente nelle richieste che richiedono autenticazione. Il logout è immediato e consiste semplicemente nella rimozione del token.
Il backend espone gli endpoint necessari per la registrazione, il login, la gestione del wallet e le operazioni riservate agli admin. Il frontend offre un’interfaccia pulita e reattiva, con pagine dedicate alla registrazione, al login e alla dashboard. L’avvio del progetto è semplice: il backend gira su localhost:8080, mentre il frontend su localhost:5173.
In sintesi, questa applicazione rappresenta una base solida per sistemi che richiedono autenticazione, gestione ruoli e un’interfaccia moderna per la visualizzazione di dati personali. È progettata per essere chiara, sicura e facilmente estendibile con nuove funzionalità.
